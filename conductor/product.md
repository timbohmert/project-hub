# Project Hub

## Overview
A personal project hub that serves as a bio, a showcase for different projects, and a place to document learnings. This project is explicitly intended as a playground for exploring new technologies and over-engineering solutions for educational purposes.

## Core Features
- **Bio:** Personal introduction and background.
- **Projects:** A gallery or list describing active and past projects.
- **Learnings:** A blog or knowledge base for documenting technical insights.

## Long-term Vision
- Transition from a monolithic prototype to a modular microservice architecture.
- Use AWS as the primary cloud platform.
- Experiment with diverse technologies (different languages, databases, infrastructure patterns).
