# Specification: Frontend Shell

## Goals
- Create a modern, responsive single-page application (SPA).
- Implement the "Bio" section.
- Provide navigational placeholders for "Projects" and "Learnings".
- Ensure the build process is clean and ready for AWS S3/CloudFront deployment.

## Tech Stack
- React (TypeScript)
- Vanilla CSS
- Vite (for fast builds)

## Visual Identity
- Clean, professional, yet "alive" (using subtle animations or unique layouts).
- Mobile-first responsive design.
