# Tech Stack

## Infrastructure
- **Cloud Provider:** AWS (Primary)
- **Infrastructure as Code (IaC):** AWS CDK (Python)
- **Deployment:** Initially static/serverless, moving towards containerized microservices (ECS/EKS) or Lambda-based services.

## Frontend
- **Framework:** React (TypeScript) - *Recommended starting point*
- **Styling:** Vanilla CSS

## Future Backend Possibilities
- Languages: Go, Rust, Node.js, Python.
- Databases: PostgreSQL, DynamoDB, Graph databases (e.g., Neptune).
