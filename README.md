# Project Hub

A central repository to showcase my various software engineering projects, experiments, and creative ideas.

## Projects Included
- **Buckl**: A project for tracking and analyzing what people are saying about things.
- **GiftApp**: A utility for managing gift lists and invitations.
- **Reddit Search**: A specialized tool for searching keywords on Reddit.

## Getting Started
To run the project-hub UI locally:
1. Navigate to the UI folder: `cd project-hub-ui`
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
